Some timers run once and stop. For breaks, sets, and routines you want the same countdown to come back. Super Easy Timer repeats it for you.

## Restart automatically

Turn on restart and the timer runs the same countdown again when it ends. Good for stretches, study blocks, and work sessions.

## Add a delay between rounds

Set an optional delay so there is a pause between rounds. The timer plays the sound, waits, then starts the next round.

## Pick what happens at zero

Combine the end options to fit your routine. Flash the time, count up after zero, and restart automatically, all at once.
